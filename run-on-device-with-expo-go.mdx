---
title: "Run on Device with Expo Go"
---

Run applications directly on a physical mobile device using Expo Go to validate layouts, navigation, responsiveness, and overall user experience in real time.

## Before You Begin

Before running your application on a device, ensure that:

- Your project has been created or opened successfully
- The application has been built successfully from the Blueprint View
- Your mobile device has internet access
- Expo Go is installed on your mobile device

## Run the Application on Your Device

Open your project in Blueprint View and ensure the latest build is complete. From the preview screen, select Run App on Your Device.

A secure QR code will be generated for mobile access.

<Frame>
  ![Image](/images/image-44.png)
</Frame>

Symplr then generates a secure QR code that allows the latest version of the application to run directly on your mobile device using Expo Go.

## Install Expo Go

After clicking the button, Symplr displays the generated QR code along with instructions for connecting your mobile device.

<Frame>
  ![Image](/images/image-61.png)
</Frame>

If not already installed, download Expo Go from your device store:

- iOS: App Store
- Android: Google Play Store

The setup instructions are also displayed directly within Symplr.

<Frame>
  ![Image](/images/image-62.png)
</Frame>

<Note>
  Keep this screen open while running the application on your device. Closing the screen may stop the application from running.
</Note>

## Scan the QR Code

Open Expo Go on your mobile device and scan the QR code.

### iOS Devices

On iOS devices:

- Open the Camera or Code Scanner
- Scan the QR code displayed in Symplr
- Tap the prompt to open the application in Expo Go

<Frame>
  ![Image](/images/image-50.png)
</Frame>

### Android Devices

On Android devices:

- Open the Expo Go application
- Use the built-in QR scanner
- Scan the QR code displayed in Symplr

<Frame>
  ![Image](/images/image-49.png)
</Frame>

- Review screens and layouts for responsiveness and visual accuracy

## Troubleshooting

### QR Code Does Not Scan

- Ensure your mobile device camera permissions are enabled
- Make sure the QR code is fully visible on screen
- Verify that your mobile device has internet connectivity

### Application Stops Running

If the application stops unexpectedly:

- Ensure the Symplr preview screen remains open
- Rebuild the application if required
- Re-scan the QR code from Expo Go

## Next Steps
