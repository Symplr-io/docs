---
title: "Themes & Customization"
---

Themes & Customization allows application-wide visual styles and reusable design settings to be configured from the Theme section.

These settings help maintain consistent layouts, spacing, typography, colors, borders, and shadow styles across the application.

## Before You Begin

Before configuring themes and customization settings, ensure:

- Project is available in Blueprint View
- Application structure is configured
- You have access to Theme settings

## Open Theme Settings

From the application workspace, open the **Theme** section from the left navigation panel.

<Frame>
  ![1 Open Theme Section](/images/1-Open-Theme-Section.jpg)
</Frame>

The Theme panel provides access to Tokens, Typography, and Shadow configuration settings.

<Frame>
  ![2 Theme Panel Overview](/images/2-Theme-Panel-Overview.jpg)
</Frame>

## Configure Theme Tokens

The **Tokens** section contains reusable visual properties used across the application.

Available token categories include:

- Border
- Color
- Component
- Radius
- Space

<Frame>
  ![3 Tokens Dropdown](/images/3-Tokens-Dropdown.jpg)
</Frame>

Each token category contains reusable configuration values used throughout the application.

## Edit Token Configurations

Existing token values can be updated directly within the Tokens view.

<Frame>
  ![4 Token Configuration](/images/4-Token-Configuration.jpg)
</Frame>

Token configurations help maintain consistent styling across application screens and components.

## Add New Tokens

To create additional token configurations, use the **Add** option available within the Theme panel.

<Frame>
  ![5 Adding New Token](/images/5--Adding-New-Token.jpg)
</Frame>

New tokens can be added for different configuration types, including colors, numeric values, text values, and boolean settings.

## Configure Typography Settings

The Typography section displays configurable text styles and formatting settings used across the application.

<Frame>
  ![6 Typography Settings](/images/6-Typography-Settings.jpg)
</Frame>

Typography settings include:

- Font size
- Letter spacing
- Font color
- Font family
- Text styles and variants

These settings help maintain consistent text presentation throughout the application.

## Configure Shadow Settings

The Shadow section displays reusable shadow configurations for application components and layouts.

<Frame>
  ![7 Shadow Settings](/images/7-Shadow-Settings.jpg)
</Frame>

## Add New Shadow Configurations

Use the configuration panel to define custom shadow styles and related visual properties.

<Frame>
  ![8 Adding New Shadow](/images/8-Adding--New-Shadow.jpg)
</Frame>

Shadow configurations can include properties such as:

- Shadow color
- Blur
- Spread
- Horizontal offset
- Vertical offset

## Next Steps

After configuring themes and customization settings, continue by editing screens, configuring workflows, validating layouts, or previewing the application through Build & Run workflows.
