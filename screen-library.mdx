---
title: "Screen Library"
---

The Screen Library allows reusable application screens and layouts to be added directly from the Marketplace within Blueprint View.

## Before You Begin

Before adding screens from the Screen Library, ensure:

- Project is available in Blueprint View
- Existing application flow is configured
- You have access to screen editing features

## Open the Screen Library

From an existing screen within Blueprint View, click the **Add** option to begin adding a new screen to the application flow.

<Frame>
  ![1 Blueprint View Add Screen](/images/1-Blueprint-View-Add-Screen.jpg)
</Frame>

From the available options, select **Screen**.

<Frame>
  ![2 Select Screen Option](/images/2-Select-Screen-Option.jpg)
</Frame>

## Browse Available Layouts

After selecting **Screen**, open the **Marketplace** tab to access reusable screen layouts available within the Screen Library.

<Frame>
  ![3 Marketplace Screen Library](/images/3-Marketplace-Screen-Library.jpg)
</Frame>

Browse the available layouts and select the screen structure that matches your application requirements.

## Add a Screen to the Application

Select the required layout to add the screen to the application flow.

<Frame>
  ![4 Select Screen Layout](/images/4-Select-Screen-Layout.jpg)
</Frame>

Once added, the new screen becomes part of the application structure within Blueprint View.

<Frame>
  ![5 Added Screen in Blueprint](/images/5-Added-Screen-in-Blueprint.jpg)
</Frame>

The added screen can then be linked, configured, and customized as part of the application workflow.

## Continue Application Configuration

After adding a screen from the Screen Library, continue configuring widgets, workflows, navigation, themes, and screen-level behaviors as required.

## Next Steps

After adding screens from the Screen Library, you can continue by configuring workflows, customizing layouts, managing navigation, or validating the application through Build & Run workflows.
