## Overview

The purpose of the Symplr Screen Library is to provide the user with a collection of pre-built Screens that can be easily added to any App that they are building. As an example, imagine a Social Login Screen that enables the App user to login using their Social Network credentials such as Google & Apple. Similarly imagine a Customer Support Screen that enables the user to Chat with a support agent or log a support request.

Symplr Screen Library is a collection of such frequently used screens that bring convenience to the user. These Screens also include the back-end API integrations so that they are fully functional.
