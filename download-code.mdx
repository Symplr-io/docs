title: "Download Code"

Download the generated application source package from Symplr to continue development, testing, customization, and deployment workflows outside the platform.

The generated code package contains the application source files produced from your application structure, screens, workflows, widgets, and configuration settings defined within Symplr.

## Before You Begin

Before downloading the generated application code, ensure that:

- Your project has been created or opened successfully
- The application structure is finalized within the **Blueprint View**
- You have access to the **Developer Tools** menu
- The application is ready for code generation

### Step 1: Open Developer Tools

Navigate to the **Blueprint View** of your project.

From the top-right corner of the Blueprint View, click the **Developer Tools** dropdown icon.

<Frame>
</Frame>

The Developer Tools menu provides access to application code generation and export capabilities.

### Step 2: Configure Generation Options

Within Developer Tools, you can configure additional options before generating the application code package.

<Frame>
</Frame>

Depending on your requirements, you can:

- Include generated test cases within the exported package
- Include additional security-related protections during generation
- Connect the project to GitHub for version control workflows

These options help prepare the generated application package for testing, validation, customization, and continued development workflows.

### Step 3: Generate the Application Code Package

After configuring the required options, click the **Download** button within Developer Tools.

Symplr begins generating the application source package based on your application structure and selected configuration options.
